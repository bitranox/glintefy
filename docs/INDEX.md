## üìñ Glossary

- **MCP**: Model Context Protocol - Standardized protocol for AI model interactions
- **Orchestrator**: Main server that coordinates sub-servers
- **Sub-server**: Specialized agent for specific analysis/fix tasks
- **Evidence-Based**: All fixes verified with before/after measurements
- **3x Verification**: Run tests 3 times to detect flaky tests
- **Integration Protocol**: Standard file format for sub-server outputs
